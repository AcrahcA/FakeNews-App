# FakeNews-App
FakeNews App es una aplicación de escritorio desarrollada en Java que permite la gestión completa de noticias falsas.
El sistema permite crear fake news, asignarles un refutador responsable, y clasificarlas mediante distintos medios y categorías. Además, cuenta con un módulo de administración donde se pueden visualizar todas las noticias en una tabla interactiva con buscador, facilitando operaciones como agregar, modificar, eliminar o refutar una noticia existente.
La aplicación también incluye la gestión de refutadores, permitiendo su alta, baja y modificación, manteniendo así un control organizado de los responsables de verificar la información.
En conjunto, la herramienta simula un entorno de análisis y control de desinformación, ofreciendo una estructura clara y funcional para el manejo de noticias falsas
